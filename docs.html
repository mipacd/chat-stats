Overview/Site Map

All data is calcuated using public, archived streams on YouTube only (unless otherwise noted).

Language

Non-JP %/Rate (HoloJP): Percentage of chat messages that are not in Japanese for each month. HoloJP and DEV_IS only.

JP %/Rate (HoloID/HoloEN): Percentage of chat messages that are in Japanese for each month. HoloEN and HoloID only.

Language Summary Stats: Means and standard deviations of JP and non-JP percentages and rates for JP (including DEV_IS), EN, and ID branches.

Avg. Chat Makeup: Breakdown of chat message types in average messages per minute. Breakdown includes messages in Japanese, Korean, Russian, emotes, and messages that don't fit into any of the other categories (EN/ES/ID/etc.).

Translations

English Live Translations: Bar graphs of the average English live translations per minute and total number of English live translations. Properly tagged, in-chat translations only.

English Top Live Translated Streams: A list of all streams in a given month where the average live translation rate is greater than or equal to one live translation per minute. Sortable by generation, translations per minute, and streamer.

Users

Common User Heatmap: Heatmap of the percentage of users shared between Hololive member's chats (percentage of x-axis' chat user's in y-axis' chat). Can be filtered to a minimum of 3 or 10 messages for each common chat user, or common membership percentages.

Exclusive Chat Users: Percentage of chat users for each Hololive member that did not participate in another Hololive member's chat.

Member Counts: Number of detected chat participating members in each Hololive members chat, broken down by membership rank (badge type).

Membership Percentages: Percentage of chat users who participated in chat with a membership during the month for each Hololive member.

Stream Time & Frequency

Streaming Hour Stats: A report containing the number of streaming hours, average stream time (for streams > 5 minutes), longest stream time, and change since the previous month for each Hololive member.

Streaming Hour Monthly Trend: Line graphs showing the monthly streaming hours for each Hololive member.

Streaming Hour Consistency: Bar graphs showing streaming hour consistency in 3 month, 6 month, and 1 year timeframes. Consistency is measured by calculating the ratio of the mean of streaming hours to the standard deviation (inverse coefficient of variation).

Activity Coverage Percentages: Line graphs showing the percentage of days in a month where each Hololive member released content (streams, music, collabs, etc.). Unlike the other datasets, this data does include unarchived streams, membership streams, and collabs.
