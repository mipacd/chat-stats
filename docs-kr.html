개요/사이트 맵

이 페이지는 기계 번역되었습니다. 모든 데이터는 공개된 아카이브된 YouTube 스트림을 사용하여 계산됩니다 (별도로 언급하지 않는 한).

언어

일본어 이외의 비율 및 비율 (HoloJP): 매월 일본어가 아닌 채팅 메시지의 비율. HoloJP 및 DEV_IS만 해당.

일본어 비율 및 비율 (HoloID/HoloEN): 매월 일본어로 된 채팅 메시지의 비율. HoloEN 및 HoloID만 해당.

언어 요약 통계: JP 및 비-JP 비율과 비율의 평균 및 표준 편차. JP (DEV_IS 포함), EN, ID 지부에 대해.

평균 채팅 구성: 평균 메시지당 분당 메시지 유형의 구분. 구분에는 일본어, 한국어, 러시아어, 이모티콘, 기타 카테고리에 속하지 않는 메시지 (EN/ES/ID 등)가 포함됩니다.

번역

영어 실시간 번역: 평균 영어 실시간 번역 수와 총 영어 실시간 번역 수에 대한 막대 그래프. 적절히 태그된 인-채팅 번역만 포함됩니다.

영어 상위 실시간 번역 스트림: 분당 평균 실시간 번역률이 1 이상인 모든 스트림의 목록. 세대, 분당 번역 수, 스트리머별로 정렬 가능.

사용자

공통 사용자 히트맵: Hololive 멤버의 채팅에서 공유되는 사용자 비율에 대한 히트맵 (x축 채팅 사용자의 y축 채팅에서의 비율). 각 공통 채팅 사용자에 대해 최소 3 또는 10 메시지로 필터링하거나 공통 멤버십 비율로 필터링할 수 있습니다.

독점 채팅 사용자: 다른 Hololive 멤버의 채팅에 참여하지 않은 각 Hololive 멤버의 채팅 사용자 비율.

멤버 수: 각 Hololive 멤버의 채팅에 참여한 감지된 채팅 멤버 수, 멤버십 등급 (배지 유형)별로 분류.

멤버십 비율: 각 Hololive 멤버의 채팅에 참여한 멤버십 사용자 비율.

스트리밍 시간 및 빈도

스트리밍 시간 통계: 스트리밍 시간 수, 평균 스트리밍 시간 (5분 이상의 스트림), 최장 스트리밍 시간, 전월 대비 변화에 대한 보고서.

스트리밍 시간 월별 추세: 각 Hololive 멤버의 월별 스트리밍 시간을 보여주는 선 그래프.

활동 커버리지 비율: 각 Hololive 멤버가 콘텐츠 (스트림, 음악, 협업 등)를 게시한 날의 월별 비율을 보여주는 선 그래프. 다른 데이터셋과 달리, 이 데이터는 아카이브되지 않은 스트림, 멤버십 스트림 및 협업을 포함합니다.
