Summary

This site shows data mostly derived from Hololive stream chat logs. Data reflects public streams with chat replay available at the time of chat log retrieval.

This site will no longer be updated. Please go to the new site.

このサイトはこれ以上更新されません。新しいサイトに移動してください。
